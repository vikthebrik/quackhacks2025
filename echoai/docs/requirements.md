# EchoAI Developer Requirements

## Required Tools
- Node.js >= 18
- NPM >= 9
- Chrome Browser >= 122
- Git

## Required Packages (auto-installed from package.json)
- @google/generative-ai
- axios
- webextension-polyfill
- compromise (NLP for token reduction)
- wink-nlp or ml-lite (optional)

## Chrome Extension Requirements
- Manifest V3
- sidePanel API enabled

## Environment Variables
- GEMINI_API_KEY=your_key_here
